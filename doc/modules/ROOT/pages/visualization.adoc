= Visualization

You can use the link:https://neo4j.com/docs/python-graph-visualization/current/[`neo4j-viz`] graph visualization library to visualize your projection within a Python environment such as a Jupyter notebook or link:https://streamlit.io/[Streamlit].

See for example how you can use link:https://neo4j.com/docs/python-graph-visualization/1/reference/tutorials/gds-example/[`neo4j-viz` with GDS].
