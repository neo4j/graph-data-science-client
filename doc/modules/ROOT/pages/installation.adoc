= Installation

To install the latest deployed version of the Python client, run:

[source,console]
----
pip install graphdatascience
----

== Optional dependencies

=== NetworkX graphs

Install the optional `networkx` dependency if you plan to load xref:graph-object#networkx[NetworkX graphs]:

[source,console]
----
pip install "graphdatascience[networkx]"
----

=== Neo4j Python driver speedup extension

The `graphdatascience` library uses the Neo4j Python driver under the hood to communicate with Neo4j DBMSs.
To use the link:https://github.com/neo4j/neo4j-python-driver-rust-ext[Rust extension] for better performance, you must install the optional `rust_ext` dependency:

[source,console]
----
pip install "graphdatascience[rust_ext]"
----

Depending on your system, you may need additional installation steps.
See the https://github.com/neo4j/neo4j-python-driver-rust-ext?tab=readme-ov-file#requirements[Rust extension README] for more details.

=== OGB dataset

Install the optional `ogb` dependency if you plan to use the xref:common-datasets#ogb[OGB dataset]:

[source,console]
----
pip install "graphdatascience[ogb]"
----
